Blog name : A guideline for 'WEB DEVOPMENT' from bigginer to expert.

So if you want to be a professional web developer and looking for a guideline then I say you that you are in the right place.Now I am telling you what should be the approach for becoming a bigginer to intermediate and inter to expert/advance developer.


In web development you have 2 parts , one is Front end & another is Back end.

Front-end: The UI.

1.HTML : The markup language. This is the first thing which you have to leran.

2.CSS : For styling and coloring the web page 'cascased style sheet' or css is the next topic you have to learn must.

3.Bootstrap : This is the CSS framework, whrer some built in css libraries are existed. I suggest you to learn it after learning the HTML and CSS.
4.Javascript & Jquery : jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers.

5.Typescrit & Ajax : AJAX is a developer's dream, because you can: Update a web page without reloading the page . Request data from a server - after the page has loaded. Receive data from a server - after the page has loaded. Send data to a server - in the background.

6.Foundation : It's like a another front-end framework as like as bootstrap. If you learn it , that will be a another skill for you with a wonderful framework.

7.Angular JS : Javascript based advance front end framework

8.React : Javascript based advance front end framework

9.Vue.js : Javascript based advance front end framework

10.NPM , Json : NPN designs and builds websites for corporate and small business customers as well as for private website owners. And JSON (JavaScript Object Notation) is a lightweight data-interchange format. It is easy for humans to read and write. It is easy for machines to parse and generate.

Back-end:The Srverside concept.

Attention please!!!, for working with the back end , back end means the serverside working concept, you must have to learn a web server side programming language. I suggest you at first learn php or python. Around 50% of todays website is run on the php even facebook. On the otherhand Python is the most popular language in 2018/19.. And peoples are coming to the python. But in my opinion learn php first , or if you want learn python no problem. So here is a priority list for you.

1.Php : A server side web programming language.

*Data structure and Algorithm. After learn a programming language , now you have to grow up your coding knowledge. So , without knowing the basic data structure , you can't servive in the back end developing world. The basic data-structure and algorithm as like as: Array , Stack , Queue ,List/linked list, map , Binary search , BFS , DFS , sorting algorithm like merge sort , quick sort etc.. , Dijkastra. I will right a blog about these as like as: "The developers must need data structure and algorithm" as soon as possible.


2.Laravel : The framework of Php. Other framework also existed such as: CodeIgniter , Symphony , Cakephp , but i think should leran laravel first.

3.Python :Programming language.

4.Django :The framework of Python. Other framework also existed such as: Fask , Pyramid , Web2py... , but i think should leran Django first.

5.Node Js : Javascript serverside programming language.

6.Express :The framework of Node Js. Other framework also existed such as: AdonisJs , MeteorJs , NestJs , but i think should leran Express first.

7.API , http , SaaS : Application programming interface , http request , cloud computing what's are the some extra and advance skill for a developer

With these, from the beggining of your journey you should learn and use to use the git and github

After this you should learn about advance Database. These may are..

1.Mysql / Sqlite
2.NoSQL (MongoDB)
3.PostGre
4.Oracle

*And to become an expert web developer you must have a good knowledge in web security..*
